# Feature Expectations Matrix - Subscription Management SaaS Platform

**Purpose:** This document defines detailed expectations for what each of the 10 BMAD agents should contribute to the key features of the Subscription Management SaaS Platform. Use this as the reference for scoring agent contributions.

**Legend:**

- **MUST Address**: Core responsibilities that are non-negotiable for this agent
- **SHOULD Address**: Important contributions expected from this agent's domain
- **BONUS If Addresses**: Advanced or cross-domain insights that add exceptional value

## Scoring Guidance

When evaluating agent contributions against this matrix:

1. **MUST Address (40 points)**: Award full points only if ALL items are comprehensively covered
2. **SHOULD Address (30 points)**: Award proportional points based on coverage depth
3. **BONUS If Addresses (30 points)**: Award full credit for any advanced contributions

**Quality Criteria:**

- **Specificity**: Vague mentions don't count; require concrete details
- **Actionability**: Recommendations must be implementable
- **Domain Expertise**: Must demonstrate agent's specialized knowledge
- **Cross-Feature Coherence**: Contributions should align across all features

**Disqualifiers:**

- Generic advice that any agent could give
- Copy-pasted content without customization
- Lack of technical depth or business acumen
- Missing critical domain-specific concerns
